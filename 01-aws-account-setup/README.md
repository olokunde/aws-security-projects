# AWS Account Setup & Hardening

## Overview
Created and secured a new AWS account following day-one security
best practices before building any resources.

## Services & Concepts
AWS account root user · Multi-Factor Authentication (MFA) ·
Free Tier · billing alerts

## What I Built
- Created an AWS account and secured the root user with MFA
- Enabled Free Tier billing alerts to prevent unexpected charges
- Verified account activation and reviewed Free Tier limits

## Challenge
Hit a signup error during account creation and resolved it by
troubleshooting and retrying the registration.
